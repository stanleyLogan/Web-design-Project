Overview of the Course Registration System

The Course Registration System is designed to facilitate student registration, course management, and instructor handling within a web-based interface. The system connects to a relational database to store student, course, and instructor data.

The system has a simple user interface made up of HTML forms that interact with the backend via server-side logic. The web interface handles user requests, such as enrolling students, adding instructors, and dropping courses, while the backend processes and stores the data in a database.

Summary of the Contents of Each Page

  • Home Page (index.html): The main landing page with links to other sections of the registration system.
  • Enrollment Page (Enrollment Page.html): A form for students to enter their details and enroll in courses.
  • Add Instructor (addinstructor.html): A page for adding new instructors to the system.
  • Add Course (addcourse.html): A form to add new courses to the system.
  • Register for Courses (registercourse.html): A page where students can select courses to register for.
  • Drop Course (dropcourse.html): A form that allows students to drop a course they are enrolled in.
  • User Manuals (manuels.html): A page linking to both user and programmer manuals.

Description of all Form Elements

The following is a description of the form elements used on each page:

Enrollment Page (Enrollment Page.html)

  • First Name (id="First Name"): A text input field for entering the student's first name.
  • Last Name (id="Last Name"): A text input field for entering the student's last name.
  • Student Year (id="Student Year"): A dropdown select element to choose the student's year (2022, 2023, etc.).
  • Student Major (id="Student Major"): A text input field for entering the student's major.
  • Student Email (id="Student Email"): A text input field for entering the student's email.
  • Submit Button (id="submit-enrollment"): A button that triggers the form submission for student enrollment.

Add Instructor Page (addinstructor.html)

  • Instructor First Name (id="instructorFirstName"): A text input for the instructor's first name.
  • Instructor Last Name (id="instructorLastName"): A text input for the instructor's last name.
  • Instructor Department (id="instructorDept"): A text input for the department of the instructor.
  • Instructor Email (id="instructorEmail"): A text input for the instructor's email address.
  • Rank (id="rank"): A select element to choose the instructor's rank (Instructor, Assistant Professor, etc.).

Add Course Page (addcourse.html)

  • Course Name (id="courseName"): A text input for entering the name of the course.
  • Course Prefix (id="coursePrefix"): A text input for the course prefix (e.g., "CS" for Computer Science).
  • Course Number (id="courseNumber"): A text input for the course number (e.g., "101").
  • Course Section (id="courseSection"): A text input for the course section number.
  • Enrollment Cap (id="enrollmentCap"): A text input for setting the enrollment cap for the course.
