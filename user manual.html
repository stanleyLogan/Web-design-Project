Overview of the Course Registration System

The Course Registration System is designed to facilitate student registration, course management, and instructor handling within a web-based interface. The system connects to a relational database to store student, course, and instructor data.

The system has a simple user interface made up of HTML forms that interact with the backend via server-side logic. The web interface handles user requests, such as enrolling students, adding instructors, and dropping courses, while the backend processes and stores the data in a database.

Summary of the Contents of Each Page

  • Home Page: The main landing page with links to other sections of the registration system.
  • Enrollment Page: A form for instructors to enroll a new student.
  • Add Instructor: A page for adding new instructors to the system.
  • Add Course: A form to add new courses to the system.
  • Register for Courses: A page where students can select courses to register for.
  • Drop Course: A form that allows students to drop a course they are enrolled in.
  • User Manuals: A page linking to both user and programmer manuals.

Detailed descriptions

Here is a detailed description for each page

Home Page

There is no forms on this page. Only a navigation bar on the left that has a link to each page within the system.

Enrollment Page

The enrollment page will allow an instructor to add a student to enrollment

Layout

This page has five different forms.

  • First name: only charaters are acceptable(Example: John)
  • Last name: Also only charaters are acceptable(Example: Smith)
  • Student year: Select the drop down box to select the year of the student.
  • Student Major: Just like the first and last name fields, you may only input charaters into this field(Example: Computer Science)
  • Student Email: Here is where you input the student's email. You should only insert charaters with the @ symbol.(Example: something@email.com)

Click the submit Enrollment button to save the student into the enrollment system.

Add Instructor

This page allows for instructors to be added to the system. Like the Enrollment page, it has you put in the Instructor's information

Layout

This page has five differnt forms to input information.

  • Instructor First Name:Like the enrollment page, acceptable inputs are characters only.(Example: John)
  • Instructor Last Name: Like the other forms only characters are allowed.(Example: Smith)
  • Instructor Department: Enter the department the instructor is in. Only characters for this form as well.(Example: Chemistry)
  • Instructor Email:Here is where you input the email for the instructor. You may use special characters in the form. (Example: something@email.com)
  • Instructor Drop Down: use this dropdown box to select the rank for the instructor

Click the Submit button to input the instructor into the system.

Add a Course

You can add a course with everything you need to add a course into the system.

There are 14 forms that you can enter into

  • Semester: This is only a dropdown to select a semester(Fall, summer, spring)
  • Year:Add the year this course is offered. You may only input numbers into this form.(Example:2025)
  • Course Prefix: Add the prefix for the course into this form. This will only accept characters(Example: CSCI)
  • Course Number: Add the course number into this form. This will only accept numbers(Example: 101, 300, ect)
  • Course Section: Enter section for the course into this form. This will only accept numbers(Example: 01, 02, ect.)
  • Course name: Enter the name of the course into this form. This will only accept characters(Example: Programming 1)
  • Room: Enter the room and the number of the room into this form. This will accept both characters and numbers.(Example: Selby 239)
  • Time: The next four dropdowns are used to select the times for the course. Use the dropdown to select the time the course will be held.
  • Credit Hours: Use this form to input the amount of credit hours the course offers. This will only accept numbers(Example: 3)
  • Instructor First Name: Use this form to enter the first name of the Insturctor hosting the course. This will only accept characters(Example: John)
  • Instructor First Name: Use this form to enter the last name of the Insturctor hosting the course. This will only accept characters(Example: Smith)
  • Enrollment Cap: Enter the amount of students allowed in this course here. This will only accept numbers(Example: 28)

Use the submit button to add the course into the system.

Register for Courses

You can use this page to register a student into a course. You can enter all the information needed into this page.

Layout

There are seven forms to enter information into.

  • Student First Name: Enter the student's first name here.
  • Student Last Name: Enter the student's last name here
  • Semester: use the dropdown to select the semester the student is taking the course.
  • Year: Enter the year the student is taking course. Only numbers is accepted here(Example: 2025)
